The idea behind this exercise is to take an existing unsorted list, and:
a) add items to the list using the provided field. b) allow the user to use the Enter Key, or Enter Button to enter their item.
c) delete items from the list by clicking a delete button, and
d) line-through any finished item, leaving it on the list.

The most important goals of this exercise is to:
a) do all of this from within the JavaScript file ONLY,
b) without touching the HTML, and
c) without introducing any "bad practices" to my code.


This was a very time-consuming exercise because every example, I found online,
even the student examples provided as solutions, either heavily modified the HTML,
deleted the provided list entirely, or used code that is considered "bad practice"
by Andrei and the JavaScript community (i.e., using "onclick" and/or "innerHTML).
